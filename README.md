# Word Searcher

A French/English word lookup tool for games like Word Bomb. Type a string of letters and instantly see every word containing it, with sorting, filters, and favorites. Runs entirely in the browser, no server or install needed.

## Features

- Instant search as you type, anywhere on the page
- French and English dictionaries
- Sort by length, frequency, or random
- Advanced filters: length range, vowels/consonants only, no repeated letters
- Favorites saved between sessions
- Export results to a `.txt` file

## Usage

1. Open `index.html` in a browser.
2. Pick a language and a sort mode.
3. When prompted, select the matching dictionary file (`wordlist_fr.txt` or `wordlist_eng.txt`).
4. Type to search.

All data stays local, nothing is sent anywhere.
